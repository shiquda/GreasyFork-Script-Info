# GreasyFork-Script-Info

## 利用 Github action 自动爬取[Greasy Fork](https://greasyfork.org/zh-CN)上存在的脚本信息，输出 [csv](https://github.com/shiquda/GreasyFork-Script-Info/blob/master/script_info.csv) 文件，包括：

- ID
- 名称
- 描述
- 作者
- URL
